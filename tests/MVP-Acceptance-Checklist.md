# FocusLens MVP 验收清单

## 主流程

1. 全屏录制 5 分钟后停止，生成 `project.json`
2. 编辑页调整 trim/aspect/cursor highlight 并保存
3. 导出页启动 MP4 导出并接收 `export/progress` 事件

## 指标

1. 连续录制 30 分钟无崩溃
2. A/V 偏移绝对值 <= 100ms
3. 平均掉帧 <= 2%，峰值 <= 5%
4. 导出样例集 20 条成功率 >= 95%

## 异常

1. 模拟无写入权限，导出失败并提示
2. 模拟磁盘空间不足，导出失败并提示
3. 导出失败后可直接点击重试
4. 录制中断后重启可在恢复列表看到项目

## 平台矩阵

1. Windows：系统音频可录制
2. macOS：系统音频不可用时提示降级并可继续
